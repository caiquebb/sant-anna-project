Serviço de Manutenção e Segurança Usuários, Menus, Parâmetros e Controle de Acesso.

Sant Anna é um serviço de manutenção e segurança que controlara o cadastro de usuários, menus, parâmetros e controle de Acesso. Todos os serviços AGV deverão utlizar o mesmo para o seu controle de manutenção e segurança.

Fluxograma

Manutenção

Usuários

Realiza a manutenção do cadastro de usuários. Na tela inicial será exibida uma lista com todos os usuários cadastrados, com opção de filtros e botões para inclusão, alteração e exclusão.

Regras de Negócio

  • Lista usuários cadastrados com as informações básicas (nome, e-mail)
  • Opçéos de filtros por nome e e-mail.
  • Botão para inclusão de Novo Registro.
  • Coluna e ação para editar e excluir um registro.
  • Tela para Inclusão ou Alteração com os campos nome, e-mail, senha. Incluir área para a manutenção de perfis. Incluir área para a manutenção de acessos do usuário.

Regras de Negócio

  • Lista perfis cadastrados com as informações básicas (nome)

Menus

Realiza a manutenção do cadastro de menus. Na tela inicial será exibida uma lista com todos os menus cadastrados, com opção de filtros e botões para inclusão, alteração e exclusão.

Regras de Negócio

Parâmetros

Realiza a manutenção do cadastro de parâmetros. Na tela inicial será exibida uma lista com todos os parâmetros cadastrados, com opção de filtros e botões para inclusão, alteração e exclusão.

Regras de Negócio

  • Lista de parâmetros cadastrados com as informações básicas (controle, tipo, empresa, grupo ou usuário, parâmetro, satus).
  • Opçéos de filtros por controle, tipo.
  • Botão para inclusão de Novo Registro.
  • Coluna e ação para editar e excluir um registro.
  • Tela para Inclusão ou Alteração com os campos controle, tipo, empresa, grupo ou usuário, parâmetro, satus. O parâmetro só podera ser vinculado a um grupo ou usuário, não permitir escolher as duas opções.

Controle de Acesso

Solicitação

Realiza a solicitação de acesso para um determinado usuário. Na tela inicial será exibido um formulário para a inclusão de uma solicitação e uma lista com todas as solicitações cadastradas para aquele usuário, com opção de exclusão.

Regras de Negócio

  • Formulário para a inclusão da solicitação com os campos usuário, perfil e menu.
  • Lista com todas as solicitações em aberto do usuário selecionado no formulário.
  • O Solicitante deve escolher entre solicitar um perfil ou um menu, nunca os dois na mesma solicitação.
  • Coluna de ação para excluir uma solicitação.

Mockup

Aprovação

Realiza a aprovação de acesso para um determinado usuário. Na tela inicial será exibida uma lista com todas as solicitações pendentes de aprovação para o usuário logado, com opção de aprovar ou recusar.

Regras de Negócio

  • Lista com todas as solicitações em aberto do usuário logado.
  • Coluna de ação para aprovar ou reprovar uma solicitação.
